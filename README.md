# Teq::Scraper

This is a basic scraper that works on JS framework sites

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'teq-scraper'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install teq-scraper

## Usage

```
ruby lib/teq/teqscraper.rb
```
